# AppBattleship

This project was generated with [Angular CLI](https://github.com/angular/angular-cli) version 16.1.8.

In order to run this application follow the steps below:

1) git clone the repository "https://github.com/fabiohenrique86/app-battleship.git"

2) run npm install in the root of application folder

3) run ng serve in Terminal. It should be listening in the http://localhost:4200/
